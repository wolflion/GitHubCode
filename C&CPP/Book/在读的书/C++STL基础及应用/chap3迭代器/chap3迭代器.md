#### 3.3 进一步理解迭代器
+ 迭代器把容器和算法关联起来。
+ 迭代器思维是编制通用泛型算法发展的必然结果，算法通过迭代器依次访问容器中的元素。
+ STL标准模板库编程的基本步骤
	+ 形成容器元素
	+ 取出所需要的迭代指针
	+ 调用通用算法
