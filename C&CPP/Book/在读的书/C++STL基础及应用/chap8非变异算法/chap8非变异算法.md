##  内容
####  8.1 循环
+ for_each
	+ 遍历容器元素，对每个元素执行相同的函数操作

####  8.2 查询
+ find
	+ 在单迭代器序列中找出某个值每一次出现的位置
+ find_if
	+ 在单迭代器序列中找出符合某谓语的第一个元素
+ find_first_of
	+ 在双迭代器序列中找出子序列中某元素第一次出现的位置
+ adjacent_find
	+ 在单迭代器序列中找出第一次相邻值相等元素的位置
+ find_end
	+ 在双迭代器序列中找出一子序列最后一次出现的位置
+ search
	+ 在双迭代器序列中找出一个子序列第一次出现的位置
+ search_n
	+ 在单迭代器序列中找出一个值连续n次出现的位置

####  8.3 计数
+ count
	+ 在序列中统计某个值出现的次数
+ count_if
	+ 在序列中统计与某谓词(表达式)匹配的次数

####  8.4 比较
+ equal
	+ 两个序列中的对应元素都相同时为真
+ mismatch
	+ 找出两个序列相异的第一个元素
